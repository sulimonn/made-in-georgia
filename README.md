This is a React.js application showcasing products or services made in Georgia. It is built with Vite for fast development and optimized builds.

Tech Stack
----------
- React.js
- JavaScript (ES6+)
- Material-UI (optional)
- Redux Toolkit (optional)

Getting Started
---------------
Prerequisites:
- Node.js (v16 or higher)
- npm or yarn

Installation:

1. Clone the repository:
   git clone https://github.com/your-username/made-in-georgia.git

2. Navigate into the project folder:
   cd made-in-georgia

3. Install dependencies:
   npm install
   # or
   yarn install

Development:

Start the development server:
   npm run dev
   # or
   yarn dev

Visit http://localhost:3000 to view the app.

Production Build:

To create an optimized production build:
   npm run build
   # or
   yarn build

Preview Production Build:
   npm run preview
   # or
   yarn preview
